## How Can I Contribute?

### 1. Reporting Bugs or Issues
- Use the GitHub Issues tracker.
- Provide a clear title and detailed description.
- Include:
  - Steps to reproduce
  - Expected vs. actual behavior
  - Screenshots or logs (especially criticality alert logs)
  - Device/Android version
  - Any perceived metaphysical side effects
- Label as `bug`, `criticality-incident`, or `soul-glitch` if appropriate.

### 2. Suggesting Enhancements / New Features
- Open a new Issue with the `enhancement` or `metaphysical-upgrade` label.
- Describe the feature in detail:
  - What problem does it solve?
  - How does it improve soul tracking / aura analysis / reincarnation mapping?
